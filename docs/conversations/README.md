# 会話履歴

このディレクトリには、プロジェクト開発中の会話履歴や議論のログを保管します。

## ファイル命名規則

以下のような形式でファイルを作成することを推奨します：

- `YYYY-MM-DD_トピック名.md` - 日付ベース
- `topic_トピック名.md` - トピックベース

## 例

- `2025-12-28_初期セットアップ.md`
- `topic_コマンド設計.md`
- `topic_UI改善案.md`

## テンプレート

新しい会話ログを作成する際は、以下のテンプレートを使用できます：

```markdown
# [トピック名]

**日付**: YYYY-MM-DD
**参加者**: [名前]

## 概要

[この会話の目的や背景]

## 議論内容

### [トピック1]

[内容]

### [トピック2]

[内容]

## 決定事項

- [決定事項1]
- [決定事項2]

## 次のアクション

- [ ] [タスク1]
- [ ] [タスク2]

## メモ

[その他のメモ]
```
